# Hakkimda
Yazılım, sistem tasarımı ve oyun geliştirme alanlarında aktif olarak projeler geliştiren bir geliştiriciyim. Farklı programlama dillerini kullanarak gerçek, çalışan sistemler ve ürünler oluşturuyorum. Yapay zekayı projelerime entegre ederek akıllı, ölçeklenebilir çözümler geliştirmeye odaklanıyorum.
